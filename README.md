# IEEE UI/UX Design Workshop Website

A modern, responsive landing page developed for the IEEE Student Branch UI/UX Design Workshop.

## Features

- Modern UI/UX design
- Glassmorphism interface
- Animated landing page
- Sliding hero background
- Interactive torch effect
- Floating sparkles
- Aurora background
- 3D hover cards
- Animated statistics
- Workshop timeline
- Mentor section
- Workshop highlights
- FAQ section
- Registration popup
- Scroll progress bar
- Mouse spotlight
- Responsive design
- Mobile friendly

## Technologies

- HTML5
- CSS3
- JavaScript
