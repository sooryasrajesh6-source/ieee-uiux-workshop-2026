Workshop Overview

This workshop helps students understand how digital products are planned, designed, tested, and presented before development.

Design With Purpose

Participants will learn how to create clean, useful, and user-friendly interfaces. The workshop focuses on practical design thinking, user flow planning, and prototype creation.

Professional design workflow
Hands-on wireframing practice
Prototype creation and presentation
Useful for resume, portfolio, and LinkedIn

What You Will Learn

A structured learning experience for students interested in design, development, and digital products.

UI

Interface Design

Learn visual hierarchy, layout structure, spacing, alignment, color usage, and typography.

UX

User Experience

Understand user needs, usability, user journeys, information flow, and design decisions.

WF

Wireframing

Create low-fidelity layouts to plan screens, sections, and user interactions clearly.

PT

Prototyping

Build interactive prototypes that demonstrate product flow and user actions.

Workshop Schedule

A focused two-day structure for learning and practical implementation.

Day 1

Introduction to UI/UX

Basic concepts of user interface design, user experience, product thinking, and design process.

Day 1

Design Principles and Wireframes

Learn layout, typography, color, spacing, and create simple wireframes for an application.

Day 2

Prototype Development

Create a modern interface and convert it into a clickable prototype.

Day 2

Review and Showcase

Improve the final design, review usability, and present the completed prototype.

No. This workshop focuses on design thinking, wireframing, and prototyping.

Yes. Participants will create a simple prototype during the workshop.
